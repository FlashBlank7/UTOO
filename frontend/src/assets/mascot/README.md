# UTOO Mascot Assets

These mascot images were generated for this project as original, family-friendly UI assets.

- The seven weekday PNGs use the same character, canvas size, and transparent background.
- They are intended as the front-end fallback when Live2D is unavailable, disabled, or not yet licensed.
- The current fallback style is adult-coded and lightly erokawaii, while staying family-safe: no nudity, no exposed intimate body parts, no transparent underwear, and no strongly sexualized pose.
- Third-party Live2D models and public sprite packs were used as research references only; no third-party character asset is embedded here.
- Keep future replacements tasteful, adult-coded, and readable at small UI sizes.
