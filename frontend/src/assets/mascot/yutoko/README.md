# Yutoko Mascot Iteration Archive

Yutoko is the original UTOO mascot. This folder keeps local iteration artifacts so future visual changes do not live only in generated build output.

- `exports/` contains the current transparent PNG exports.
- `prompts/` contains the design prompt and safety/brand notes.

Brand boundary:

- Use UTokyo-inspired color and campus motifs only.
- Do not use the official University of Tokyo logo, logotype, symbol mark, emblem, or readable official university text without permission.
- The current badge is an original UTOO double-ginkgo badge, not a university mark.

Style boundary:

- Adult-coded, cute, lightly sensual, and tasteful.
- No nudity, exposed intimate body parts, transparent underwear, explicit sexual motion, or underage-coded sexualization.
