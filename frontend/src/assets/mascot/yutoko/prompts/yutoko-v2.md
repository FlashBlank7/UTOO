# Yutoko v2 Prompt And Notes

Purpose: strengthen Yutoko as the UTOO student-community mascot while keeping the site independent from official University of Tokyo branding.

Prompt direction:

```text
Create an original adult-coded anime catgirl mascot named Yutoko for UTOO, an independent UTokyo student community. Keep her cute, polished, slightly erokawaii, and family-safe. Every weekday outfit must include a clear ginkgo leaf motif, UTokyo Blue #0B8BEE, Yellow #FFCD00, and a small original UTOO double-ginkgo badge. Add campus-life details such as lab ID cards, research folders, commuting bags, Akamon-red charms, cafe apron details, hoodie/tote accessories, and light wafuu cardigan details. Do not use the official University of Tokyo logo, logotype, symbol mark, emblem, or readable official university text. Transparent PNG, consistent character, consistent canvas, readable at 96px-180px UI sizes.
```

Local v2 implementation notes:

- The current committed v2 assets were locally enhanced from the prior transparent PNG exports.
- Each weekday PNG now has at least one visible ginkgo leaf and a stronger blue/yellow identity accent.
- The badge is original and intentionally avoids official UTokyo mark geometry.
